# Stellaris MOD - hamusElectionNoInfluence
このリポジトリはParadox Interactive社のStellarisというゲームのMOD、hamusElectionNoInfluenceのコードを管理するものです。 

#  MOD概要
「民主主義」、「寡頭制」の任意の候補を統治者に選ぶ際に消費する影響力を0にするチートMODです。
